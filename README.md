# mutations
this python code is used to create a visual graphical representation of the mutations present when comparing more than two genomic sequences.
PV361318 - is used as a reference for PX063729 and PX471996 .
find_mutations.py file is used for allignment to the reference sequence.
plot_muattaions.py file is used to identify the mutations and plot a graph to represent the mutation at a certain genomic segments.
this work is done in linux terminal in conda environment and packages used are mafft, matplotlib.
